Heart Disease Prediction using Machine Learning

This project aims to predict whether a person has heart disease based on medical attributes using Machine Learning techniques.

Technologies Used

- Python
- Pandas
- NumPy
- Matplotlib / Seaborn
- Scikit-learn

Machine Learning Algorithms

- Logistic Regression
- K-Nearest Neighbors (KNN)
- Random Forest Classifier

Project Description

The dataset was first preprocessed by handling missing values and performing exploratory data analysis. Different machine learning algorithms were trained and evaluated to find the best model for predicting heart disease.

Result

The model achieved good prediction accuracy and helps in identifying potential heart disease risk based on patient health parameters.

Conclusion

Machine Learning can be effectively used in healthcare to assist in early prediction and diagnosis of heart disease.
